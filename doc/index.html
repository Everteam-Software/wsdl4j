Introduction

WSDL4J is the reference implementation of Java APIs for WSDL (JWSDL). JWSDL is a Java Specification Request (JSR) being developed under the Java Community Process.

A javax.xml.namespace.QName implementation is also provided.

This work is being made available under the IBM Common Public License.

Please click here for the JWSDL 1.2 Maintenance Release specification.

Please click here for the WSDL4J API JavaDocs.

Requirements

WSDL4J has the following requirements:

  • A JAXP compatible, namespace-aware XML parser; Apache Xerces works fine
  • JDK1.4 or greater.
